# Source Registry Changelog

Historical engineering notes for every version-bumping change to [`data/sources.json`](data/sources.json) -- source additions, fixes, and registry-health findings, in order. This file exists purely for engineering history; it is never fetched by any page or app (unlike `data/sources.json` itself, which the live [`/sources/`](https://equalinformation.com/global-intelligence-site/sources/) page downloads in full on every visit -- keeping this changelog separate keeps that payload lean).

- Machine-readable registry of the world's important news media, consumed by the Global Intelligence pipelines and displayed on the site.

- RSS URLs are best-known candidates and are UNVERIFIED until the pipeline build validates each one; prefer RSS/official APIs over scraping.

- Pipelines must summarize in our own words and attribute/link to sources — never republish article text.

- state_affiliated=true marks government-funded or state-controlled outlets; the AI curation layer must weight/flag these for bias.

- This is v0.1 — coverage will grow toward all countries/regions.
